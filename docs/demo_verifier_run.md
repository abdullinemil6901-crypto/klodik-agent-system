# Живой прогон: adversarial-проверка фактов

Разбор реального запуска конвейера `finder → verifier → synthesizer` (радар рыночных возможностей, июнь 2026). Это тот самый прогон, который упомянут в README строкой «из 6 сырых сигналов выжило 3» — здесь показано, что именно и почему отсеялось.

## Постановка

- **2 finder-агента параллельно**, углы: «новые технологии/инструменты» и «рыночный спрос/деньги».
- **1 verifier** — скептик. Презумпция: сигнал неверен, пока не подтверждён вторым независимым источником. Отдельно проверяет: свежесть дат, реальность цифр, конфликт интересов источника.
- **Синтез** — отбор выжившего под контекст пользователя из слоя памяти.

## Вход: 6 сырых сигналов от finder-агентов

Каждый сигнал на входе выглядел убедительно — со ссылкой, цифрой и датой.

## Вердикты verifier

| Сигнал | Вердикт | Основание |
|---|---|---|
| Рост спроса на навык «+938%» | отсев | цифра из маркетинга фриланс-площадки, продающей этот же навык — конфликт интересов, независимого подтверждения нет |
| Voice-агенты как «золотая жила» | отсев | хайп-волна: восторженные обзоры есть, подтверждённого спроса нет |
| Прайсы на генерацию видео | исправлено | у finder $0.03/сек; по прайс-листу вендора реально $0.05–0.15/сек |
| Дата релиза модели | исправлено | finder принёс 31-е число, по первоисточнику — 22-е |
| Кейс дохода «$8K/мес» | приземлён | единичный кейс-бестселлер; по независимым данным площадок реалистичный старт $500–2000/мес |
| Рост ниши AI-video на фриланс-бирже (+329%) | подтверждён | verifier сам нашёл второй независимый источник (статистика Upwork), которого не было ни у одного finder |

Итог: из 6 сигналов выжило 3 (одно подтверждено независимо, два исправлены до реальных значений). Одиночный поиск выдал бы все 6 как «факты».

## Что показал прогон

1. **Verifier не только режет, но и добавляет.** Второй независимый источник по единственной подтверждённой цифре нашёл именно скептик, а не искатели — у них другая задача и другой промпт.
2. **Типовые ловушки предсказуемы:** маркетинговые цифры заинтересованных площадок, устаревшие даты, единичные кейсы, выдаваемые за норму. Под них verifier получает явные инструкции — «презумпции недоверия» в промпте.
3. **Цена паттерна измерима:** 3 суб-агента, ~128k токенов на прогон — те самые ~15× против одиночного поиска. Отсюда правило роутинга: конвейер только на параллелизуемых задачах, рутины стартуют с 1 прогона в день.
