# vault-template — скелет файловой памяти

Обезличенная копия структуры рабочего vault. Как использовать:

1. Скопировать папку в рабочую директорию (например `~/my-agent/vault/`) и открыть её в Obsidian как vault (или работать любым редактором — это обычный markdown).
2. Заполнить `00 — Профиль/` под себя: кто вы, стиль ответов, обязательные правила.
3. В `CLAUDE.md` корневой рабочей папки сослаться на vault — с этого момента каждая сессия Claude Code стартует, уже зная контекст.
4. Дальше vault наполняется сам: агент пишет журнал фидбэка и статусы проектов по ходу работы.

Принцип: не плодить новые файлы под каждый факт — обновлять существующие. Один файл = одна зона ответственности.
